# Galaxy Defiance - A Godot game

This is a space shooter game based on a [tutorial series](https://www.youtube.com/playlist?list=PL9FzW-m48fn09w6j8NowI_pSBVcsb3V78) by Heartbeast.
This project was made using Godot 4.5.1. All art assets were created by me using Krita.
